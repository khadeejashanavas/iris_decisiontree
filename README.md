Iris K-Nearest Neighbors (KNN) Classifier with Streamlit Frontend
Project Overview

This project is a machine learning web application that classifies Iris flowers into one of three species using the K-Nearest Neighbors (KNN) algorithm. The frontend is built with Streamlit, allowing users to interactively input flower measurements and get predictions instantly.

The Iris dataset contains measurements for:

Sepal Length
Sepal Width
Petal Length
Petal Width

The model predicts one of the following species:

Iris-setosa
Iris-versicolor
Iris-virginica
Features
Train a KNN classifier using the Iris dataset
Interactive web interface using Streamlit
Real-time prediction of Iris species
Simple and beginner-friendly project structure
Fast and lightweight application
Technologies Used
Python
Scikit-learn
Pandas
NumPy
Streamlit
